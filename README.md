# Claude Bridge Local

Claude Code を OpenRouter / Ollama / GLM (Z.ai) で動作させるプロキシブリッジ。

## セットアップ

1. **config.json を作成**
   ```bash
   copy config.json.example config.json
   ```
2. **config.json を編集** - APIキーを設定
3. **起動**: `start-bridge.bat` を実行

## 必要な環境

- Node.js 18+
- (Ollama使用時) Ollama インストール済み
