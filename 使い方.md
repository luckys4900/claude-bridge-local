# Claude Bridge - ローカルGPU運用ガイド

## これは何？

Claude Code が通常使う Anthropic API（有料）の代わりに、
**自分のPC（GPU）で動くOllamaモデル**を使うための仕組みです。

```
通常:    Claude Code → Anthropic API（トークン消費・課金）
Bridge:  Claude Code → claude-bridge → Ollama（自分のGPU・無料）
```

## 起動方法

### 方法1: メニューから選ぶ（推奨）
`start-bridge.bat` をダブルクリック → モデルを番号選択

### 方法2: 直接起動
- `start-qwen3-30b.bat` → 高品質（30Bモデル・重い）
- `start-llama3-fast.bat` → 軽量・高速（2GBモデル）

## 使えるローカルモデル

| モデル名 | VRAM目安 | 特徴 |
|---------|---------|------|
| qwen3:30b | 20GB+ | 最高品質、日本語強い |
| codellama | 4GB | コード作業専用 |
| llama3.2 | 2GB | 軽量・高速 |

## 注意点

- トークン使用量の表示は正確でない（常に0など）→ 正常
- 画像アップロードは動かない
- Ollamaが起動していないと使えない（タスクトレイ確認）
- 性能はClaude Sonnet/Opusには及ばない場合がある

## コマンドラインから直接使う場合

```bash
# PowerShell または コマンドプロンプト
claude-bridge openai qwen3:30b --baseURL http://localhost:11434/v1
claude-bridge openai llama3.2 --baseURL http://localhost:11434/v1
claude-bridge openai codellama --baseURL http://localhost:11434/v1
```
